## How to Host on GitHub Pages

Follow these simple steps to host this proposal page on GitHub and access it from anywhere:

### Step 1: Create a GitHub Account
1. Go to [github.com](https://github.com)
2. Click "Sign up" if you don't have an account
3. Follow the registration process

### Step 2: Create a New Repository
1. Log in to GitHub
2. Click the "+" icon in the top right corner
3. Select "New repository"
4. Name it: `valentine-proposal` (or any name you prefer)
5. Make sure it's set to **Public**
6. Check "Add a README file"
7. Click "Create repository"

### Step 3: Upload the HTML File
1. In your new repository, click "Add file" â†’ "Upload files"
2. Drag and drop the `index.html` file or click "choose your files"
3. Add a commit message like "Add Valentine's proposal page"
4. Click "Commit changes"

### Step 4: Enable GitHub Pages
1. In your repository, go to "Settings" (top menu)
2. Scroll down and click "Pages" in the left sidebar
3. Under "Source", select "Deploy from a branch"
4. Under "Branch", select `main` (or `master`) and folder `/root`
5. Click "Save"

### Step 5: Access Your Page
1. Wait 1-2 minutes for GitHub to deploy your page
2. Your page will be available at:
   ```
   https://YOUR-USERNAME.github.io/valentine-proposal/
   ```
   Replace `YOUR-USERNAME` with your GitHub username

### Alternative: Quick Setup with GitHub Desktop
1. Download [GitHub Desktop](https://desktop.github.com/)
2. Create a new repository
3. Add the `index.html` file to the repository folder
4. Commit and push to GitHub
5. Follow Step 4 above to enable GitHub Pages
